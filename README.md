# 重要なお知らせ

現在配布しているバイナリ（プレビュー版）において、「Bing検索連携」の設定完了後、**「プライバシーとセキュリティ」→「検索」**において、**「この設定はorganizationによって管理されています」という表示が出現**し、**「検索ハイライト」がグレーアウト**する事象を確認しました。
原因としては、設定適用時にグループポリシーに書き込みを行う設定となっており、**個人用であるにもかかわらず、組織によって管理されているとみなされる**状態になっています。

なお、**このような設定は数多のWebサイトにて公開されているレジストリ設定**や、**他のWindows11設定変更ツール**においても確認され、ユーザーによっては不安を覚える方もいらっしゃるかと思われます。

次期バージョンでは設定項目の拡張・細分化のほか、**極力グループポリシーに書き込みを行わないよう仕様変更**を行い、不安の解消に努めます。

## PoletoWin-No11
Windows11の高負荷やトラブルの原因となる設定をコントロールし、安定性とプライバシーを強化します。

## 各種機能説明

| 設定項目                               | 項目詳細                                                                                                                                                                                                                   | 
| -------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | 
| Bing検索連携<br>（Explorer再起動）     | スタートメニュー・検索ボックスはデフォルトでBing連携機能がオンになっており、Webサイトを検索結果に含めるようになっています。オフにすることで、検索結果がローカルのファイルやアプリ・Windowsストアの検索結果のみになります。 | 
| フォルダ自動検出<br>（Explorer再起動） | フォルダの内容をスキャンすることにより、フォルダを開いた際に最適な項目を表示するようにしています。オフにすることでフォルダを開くごとのスキャンはなくなり高速化しますが、不便な場合もあります。                             | 
| 広告ID                                 | 「広告識別子」をオフにすることで、送信されたユーザーの情報をもとにしたパーソナライズ広告が表示されなくなります。                                                                                                           | 
| 透明効果<br>（即時適用）               | ウィンドウの透明効果をオフにすることで、少し負荷が減ります。                                                                                                                                                               | 
| タスクバー配置<br>（即時適用）         | スタートメニュー位置「中央揃え」と「左揃え」の切り替えができます。                                                                                                                                                         | 
| タスクビュー<br>（即時適用）           | 「タスクビュー」ボタンの表示・非表示が可能です。                                                                                                                                                                           | 
| 右クリックメニュー<br>（PC再起動）     | 「従来仕様」:Windows10までのメニューに戻ります。<br>「Windows11仕様」:Win11以降の新仕様になります。                                                                                                                        |
